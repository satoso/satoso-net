GitHubのpull requestに対するコメントの付け方は4種類

2014-05-17 Sat

GitHubのissue(PR)に対するコメントの付け方がいろいろあってよく分からなくなったので実験してみた。4種類ある。

  • pull requestに対するコメント:
    <username> commented
  • commitに対するコメント:
    <username> commented on <commit hash>
  • commitの特定の行に対するコメント:
    <username> commented on <commit hash> <filename>:L<line number>
  • filesの特定の行に対するコメント:
    <username> commented on the diff

すべてが入ったサンプルはこちら。
